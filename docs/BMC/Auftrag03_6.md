# Auftrag 03.6 – Ertragsquellen

- **Launch-Paket (Fixpreis):** z. B. **ab CHF 3’490** je Projekt.
- **Module/Upsells:** Mehrsprachigkeit, Loyalty, Automation (Ø +CHF ~750/Kunde).
- **Care-Plan (Abo):** z. B. CHF 79/Monat; Ziel: **60 % Attach-Rate** (stetiger MRR).
- **Workshops/Schulungen:** Tages-/Stundensätze (z. B. Conversion-Audit).
- **Prepaid-Support:** Kontingente mit Rabatt, verbessert Cashflow-Planung.

**Preislogik:** „Ab-Preis“ + modularer Aufbau = Transparenz + niedrige Einstiegshürde.
