# Aufgabe 03.1 – Kernfähigkeiten

## Bereits stark
- **Schnelle, stabile Implementierung:** Templates, Checklisten, Test-Routinen.
- **CH-Spezifika im Checkout:** TWINT/Karte, Versandprofile, Steuerregeln → weniger Abbrüche.
- **Transparente Kommunikation:** Klarer Scope, Definition of Done, Demo-basierte Abnahme.

## Gezielter Ausbau
- **Conversion-Optimierung:** A/B-Tests, Trust-Elemente, PDP-Optimierung.
- **Marketing-Automation:** Segmente, Trigger-Mails, Warenkorb-Reminder.
- **Drittsysteme:** Standards für ERP/Marktplatz-Anbindungen, wiederverwendbare Connectoren.

**Kriterien erfüllt:** kundennützig, schwer imitierbar (Prozess + Know-how), mehrfach nutzbar (Skalierung), kernnah zum Wertangebot.
