# Auftrag 03.5 – Kostenstruktur

- **Fixkosten:** Tools/Lizenzen, Marketing, Admin, Demo-Umgebungen (ca. CHF 12–20k/Jahr je nach Phase).
- **Variable Kosten/Projekt:** externe Spezialstunden, Transaktions-/Toolkosten, Support (≈ CHF 550).
- **Investitionen:** Templates, Schulungsunterlagen, Automation-Skripte, Demo-Shops.
- **Skaleneffekte:** Wiederverwendung senkt Setup-Zeit → höhere Marge.
- **Szenario:** Höhere Partnerquote senkt Capex, erhöht Opex → Marge/Projekt sinkt, Durchsatz steigt.
